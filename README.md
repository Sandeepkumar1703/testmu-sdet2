# SDET — Test Automation Framework

## Framework: Playwright + TypeScript

---

## Project Overview

A professional end-to-end test automation framework built with Playwright and TypeScript.
Covers UI testing, API testing, integration testing, CI/CD pipeline, and reporting.

---

## Tech Stack

| Tool | Purpose |
|---|---|
| Playwright | UI & API Automation |
| TypeScript | Type-safe test code |
| JSONPlaceholder API | REST API testing |
| Sauce Demo | UI test application |
| GitHub Actions | CI/CD pipeline |
| HTML Reporter | Test reporting |
